Apepeha ZNK-1

Этот проект посвящён созданию микролитражного самоходного транспортного средства с бензиновым двигателем.

Особенности модели ZNK-01:

  • Двухтактный бензиновый двигатель с рабочим объёмом 25.4см3, мощностью в 2.6 ишачиных сил

Для чего создаётся:

  • Торжественно клянёмся, что замышляем только шалость
      • Научиться конструировать самоходные ТС для проектирования картов/багги
  • Научиться делать аксессуары для ТС (спидометры, лампочки итд)
